# @accounts/mongo

_Postgres adaptor for accounts_

[![npm](https://img.shields.io/npm/v/@accounts/postgres.svg?maxAge=2592000)](https://www.npmjs.com/package/@accounts/postgres)
![MIT License](https://img.shields.io/badge/license-MIT-blue.svg)

## Note

This package is under active development.

## Install

```
yarn add @accounts/postgres
```

## Usage

```javascript
import { AccountsServer } from '@accounts/server';
import { Postgres } from '@accounts/postgres';


The users will be saved under the `users` table.
```
